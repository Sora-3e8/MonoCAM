MonoCAM
===============
Simple crossplatform camera application.
Based on GTK 4.0 and OpenCV 2. 

Supported platforms
---------------
- Linux
- Mac OS (untested)
- Windows (wip)

Requirements
---------------
- cmake
- gtk4
- opencv

Build
---------------
```
git clone https://github.com/Sora-3e8/MonoCAM
mkdir build
cd build && cmake ..
```
